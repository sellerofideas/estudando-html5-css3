Outras formatações

Codigo fonte

O comando <code> é usado para espassar os caracteris dando uma melhor visibilidade.

O comando <pre> é usado para mostrar o texto do jeito que estiver escrito, muito útil para escrever um códico mostrando sua identação.

ex: 

            
        <pre>
            <code>
                num = int(input('digite um número'))
                if num % 2 == 0;
                    print(f'O numero {num} é par')
                else;
                    print(f'O numero é impar')
            print('fim do programa')
            </code>
        </pre>

Citações sem quebra de linha

Quanto se está fazendo uma citação, é importante usar a tag <q>, ex: Como diria o pai de um amigo: <q>o computador é um burro muito rápido.<q>

Texto invertido

Usando a tag <bdo:r> a frase se inverte ficando de frente para trás.

Usando a tag <bdo:l> a frase invertida fica De frente para trás.
