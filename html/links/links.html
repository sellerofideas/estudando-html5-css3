Como funcionam os links

Os links são aquilo que liga uma pagina de um site a outra. Para criar um link usamos a tag <a>.

Atributos do hiperlink

  • <a href="https://github.com">Clique aqui!</a>

    • Seu prncipal atributo é o "href" que cria uma referencia hipertexto dando acesso a url.

  • <a href="https://github.com" hreflang="pt-br">Clique aqui!</a>

    • Outro atributo importante hreflang que permite aos buscadores saber alinguagem da pagina para onde o link leva e assim oferecer opções de tradução ao usuario.

  • <a href="https://github.com" target="_blank">Clique aqui!</a>

    • Esse atributo marca o link como "alvo" e serve para controlarmos aonde a pagina de destino do link será aberta. Caso queira abrir um conteúdo do proprio site podemos usar o modo padão sem adicionar "target" ou acrescentar o valor "_self" ao atributo. Caso o conteúdo seja externo, o ideal é usar o valor "_blank".

    Esses são alguns dos valores que podem ser acrescentados ao atributo.

    • _blank abre uma nova janela em branco.
    • _self Abre na mesma janela.
    • _top fecha todas as abas e abre a janela de destino.
    • _parent Abre dados no tamanho da janela pai.
    • nome-do-frame Quando usamos frames, nesse caso precisamos colocar o nome da janela que irá abrir.

  • <a href="https://github.com" target="_blank" rel="next">Clique aqui!</a>

    • O atributo "rel" indica para o buscador qual é a natureza do link.

    • next indica que o link é para a proxima parte do conteúdo, como a proxima pagina.
    • prev indica que o link é para a pagina anterior.
    • author indica que o link é para a pagina do autor do conteúdo.
    • external indica que o link é para um conteúdo externo.
    • nofollow indica que o link é para um site pago etc.

Usando links externos

Como você viu, links externos abrem outras abas no navegador e tem o objetivo de conduzir o visitante sem fechar seu site, ex:

Você pode usar o meu repositório no github

Você também pode acessar meu linkedin

Links internos

No caso de links internos, a janela é aberta dentro do site sem abrir outra aba,

Esta é a pagina atual do meu site, se você quiser pode acessar também segunda pagina.

Baixar arquivos

Para disponibilizar arquivos para download, basta adicionar o arquivo desejado em uma pasta e usar a tag <a> com o parâmetro "download".

Links para dowload

  • Baixar o livro em pdf

Para baixar o arquivo direto basta adicionar a tag <a href="livro/100 títulos.pdf" download="livro/100 títulos.pdf" type="aplication/pdf">Baixar direto</a>

Esse comando não é aceito em todos os navegadores.
