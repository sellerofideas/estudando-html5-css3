Formtação de texto

Principais formatações

Negrito / Destaque

Existe uma forma não semantica e uma forma semantica para destacar uma frase
A forma não semantica é adicionando a tag <b > antes e depois</b> da frase ou palavra que se quer destacar.

A forma semântica (com significado) de usar negrito é usando a tag <strong> entre a frase que se quer destacar <strong>.

Itálico / Ênfase

Existem também 2 formas de deixar o texto em ítálico, uma forma não semântica colocando <i>o texto desejado <i> entre a tag <i>.

A forma semântica é colocando o texto entre a tag <em>.

Texto marcado

Podemos usar a tag <mark> para marcar um texto em HTML.

Ou então usar <mark styles"background-colora: greenyellow;"</mark> para marcar com outras cores usando CSS.

Para mudar a cor de todas as tags <mark>, na área de configurações da pagina, dentro de <head> precisamos criar a tag:
<style> mark { backgroud-color: goldenrod; } </style> mudando a cor padrão do marcador.

Alterar o tamanho de texto

Para aumentar o texto era usado em HTML a tag <big>para aumentar</big> e <small>para diminuir </small> a letra.

Texto deletado / riscado e texto inserido / sublinhado

Para riscar um texto pode-se usar a tag <del>.

Para sublinhar um texto usando a forma semântica, usa-se a tag <ins>, para usar a forma não semântica usa-se a tag <u>.

Texto sobrescrito

Para sobrescrever um texto como: "x20" usamos a tag <sup>.

Texto Subscrito

Para subscrever um numero como H2O, usamos a tag <sub>.

-Consulte o site: https://dev.w3.org/html5/pf-summary/obsolete.html para ver quais tags não estão mais em uso.
