Codigo fonte

A tag <code> é usada quando queremos escreverr um código dentro do conteúdo, no caso de uma pagina que ensina alguma linguagem de programação, por exemplo. Esse comando serve para espassar os caracteris dando uma melhor visibilidade, mas acaba desmanchando o codigo e não mostrando como deveria ser.

A tag <pre> é usada para mostrar o texto do jeito que estiver escrito, muito útil para corrigir o problema da tag "code", pois deixa o códico mostrando sua identação.

Ex: 

                
                <pre>
                   <code>
                    num = int(input('digite um número'))
                    if num % 2 == 0;
                        print(f'O numero {num} é par')
                     else;
                        print(f'O numero é impar')
                    print('fim do programa')
                    </code>
                </pre>

Citações sem quebra de linha

Quanto se está fazendo uma citação, é importante usar a tag <q>, que vem do inglês quote e signifíca "Citar".
Ex: Como diria o pai de um amigo: <q>o computador é um burro muito rápido.<q>
