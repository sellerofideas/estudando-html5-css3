Listas em HTML

Temos em HTML 3 tipos de listas:

Listas ordenadas

São usadas para criar uma lista com hierarquia. Para criar uma lista ordenada usamos a tag <ol>. Podemos também usar parâmetros para organizar a lista em ordem alfabética maúscula, por exemplo, nesse caso adicionamos o parâmetro type="A".

  1. Tomar cafe
  2. Almoçar
  3. Jantar

O parâmetro type="a"

  1. Tomar cafe
  2. Almoçar
  3. Jantar

O parâmetro type="I"

  1. Tomar cafe
  2. Almoçar
  3. Jantar

O parâmetro type="i"

  1. Tomar cafe
  2. Almoçar
  3. Jantar

Lista não ordenadas

Listas não ordenadas são listas que não tem hierarquia. Nesse caso também podemos usar parâmetros como type="square" para mudar o demarcador da lista.

  • Pão
  • Tomate
  • Alface

Type="disc".

  • Pão
  • Tomate
  • Alface

Type="circle".

  • Pão
  • Tomate
  • Alface

Lista de definição

Essa tipo de lista serve para criar definições como um dicionário.

HTML
Linguagem de marcação para a criação de conteúdos de um site.
CSS
Linguagem de marcação para a criação do dsing de um site.
JavaScrypt
Linguagem de programação para a acriação de interatuvidade com um site.

Junção de listas 1.0

Para criar uma lista dentro da outra coloque as tags arrumando a lista como desejar, unindo uma lista organizada com números ou letras e outra com símbolos.

  1. Item
    1. Item
    2. Item
  2. Item
    • Item
    • Item
  3. Item
  4. Item
