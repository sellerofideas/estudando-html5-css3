Trabalhando com audios.

Patra trabalharmos com audios usamos a tag <audio>.

h2

Assim como as imagens, os audios precisam de versões em outros formatos par oferecermos uma melhor experiência aos usuarios. Nem todos os navegadores aceitam todos os formatos.

Os formatos de audios

Os formatos de audios mais usados são:

    • .MP3
    • .OGG
    • .WAV

É recomendado também, caso nenhuma das auternativas funcionem, colocar a função para o usuario baixar o arquivo usando a tag <a href="local-do-arquivo" type="audio/mpeg">Clique aqui para baixar a música.</a>
