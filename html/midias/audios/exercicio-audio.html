Trabalhando com audios.

Esxistem 2 formas de se trabalhar com audios:

  1. Uma é usando a tag <audio src=""></audio>.
  2. A outra é usando a tag audio + tipe (<audio> <source src="" type=""></audio>).

    3. Os formatos de audio mais compativeis são:

      • .MP3
      • .OGG
      • .WAV

É recomendado também, caso nenhuma das auternativas funcionem, colocar a função para o usuario baixar o arquivo usando a tag <a href="local-do-arquivo" type="audio/mpeg">Clique aqui para baixar a música.</a>
