Imagens em HTML para diversos formatos

O objetivo desse exercício é abrir o site me diversos formatos.

Para isso é preciso usar a tag "picture" e dentro dela usar a tag "<source media="(min-width: )" srcset="" type="image/"> e organizadas do menor para o maior.
