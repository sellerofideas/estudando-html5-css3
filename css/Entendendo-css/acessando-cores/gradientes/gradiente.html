Testando gradientes

  • Para trabalhar com as cores de fundo podemos usar a técnica de CSS estilo local (Interno na própria página) usando a área da tag "head", ou criar um padrão usando a técnica de CSS externa

Configurações.

Para configurar usando CSS local, na area da head use a tag <style>. 

    
        
  • <style>
            body {
                    background-image: radial-gradient(circle,
                     rgb(253, 253, 253), blue 70%);
                    background-attachment:
            }
        </style>
    •

    Para aplicar usando CSS externa, crie o link "link:css" na área da "head" e configure como quiser. saiba mais sobre links de CSS externo aqui.

    Você também poderá usar outros tipos de degradês como linear, de um canto ao outro, radial, etc. e também acrescentar porcentagem ás cores que escolher, se quiser mais da cor azul e menos a cor verde coloque blue 60%, green 40%.

    <p style="background-image: linear-gradient(to right, yellow 60%, green 40%);"></p>
