Identificadores (#)

Quando queremos personalizar um elemento expecífico podemos criar um identificador dentro da tag usando o atributo "id=".

Uma vez que o elemento que será personalizado foi identificado, chamamos o id, na folha de CSS ou na "head" usando "#", ex:

  1. Marcamos o elemento que queremos personalizar
    2. 
    <h1 id="principal">Identificadores </h1>
  2. Em seguida chamamos o id na folha de CSS ou na head da mesma pagina HTML

    #principal{
        font: itálic bolder 3em 'work sans', sans-serif
        color: white;
        bacground-color: blue;        
    }

Por vias de regra só podemos usar um id para cada pagina e ele sempre irá sobrepor uma classe ou configuração padrão.

Classes (.)

Uma classe em CSS possui um conjunto de caracteristicas usadas por todos os elementos identificados por ela.

Se quisermos personalizar vários elementos podemos criar classes dentro da tag desejada e usarmos o atributo "class=" e darmos um nome a classe, por exemplo: class="subtitulo".

Para configurar uma classe, na folha de CSS, chamamos o nome da classe usando um ponto, ex:


    .subtitulo{
        font: itálic bolder 3em 'work sans', sans-serif
        color: white;
        bacground-color: blue;        
    }

Podemos usar as configurações de classes para outros elementos da pagina, basta enveloparmos o elemento desejado com a tag <span> cuja finalidade é ser um container onde podemos atrubuir valores e atributos e estilizarmos seu conteúdo com CSS.

Também podemos usar mais de uma classe dentro de uma mesma tag, ex: "cass="class class2".

Também odemos adicionar uma ou mais classes junto com o "id", ex: "id="principal" class="class3".

Pseudo-classes (:)

Usamos a pseudo-classe para adicionar estados especiais dentro de de um elemento, ela é uma palavra-chave adicionada após ":" em folha de CSS, como por exemplo, ":houver" ou ":visited".

Pseudo-elemento (::)

Um pseudo-elemento CSS é uma palavra-chave adicionada a um seletor que permite que você estilize uma parte específica do elemento selecionado. Por exemplo, o pseudo-elemento ::first-line aplica o estilo apenas na primeira linha de um parágrafo..

Veja mais sobre pseudo-classes e pseurdo-elementos aqui.

Children ou filha(>)

Uma filha é uma classe dentro de outra classe, com ela podemos configurar elementos dentro de outros.
