Usando CSS para dar estilo no conteúdo do site

CSS externo

Essa técnica nos permite personalizar várias paginas de uma só vez, criando um padrão.
Basta criar um arquivo CSS usando a tag "link:CSS" dentro da "head", depois "Ctrl + clique no nome do arquivo".
No arquivo CSS é só adicionar as configurações desejadas e todas as paginas serão mudadas e padronizadas simultâniamente.

CSS interno2

Essa técnica permite mudar vários elementos dentro de uma pagina ao mesmo tempo. Ela serve para alterar elementos como titpulos, paragrafor entre outros elementos que se repetem na pagina sem a nescessidade de alterar 1 por 1.
Para usar essa técnica basta adicionar as configurações CSS desejadas dentro da "head", assim a pagina será alterada.

Capitulo 2

CSS inline

A técnica INLINE é usada caso queira alterar somente um dedeterminado elemento da pagina. Para isso, basta adicionar o codigo CSS na tag da linha desejada.
Ex: <p style="color: brown;">Técnica CSS inline</p>
