Identificadores classes, pseudo-classes e pseudo-elementos.

Identificadores (#)

Quando queremos personalizar um elemento expecífico podemos criar um identificador dentro da tag usando "id=".

Uma vez que o elemento que será personalizado foi identificado, ex: "id=principal", chamamos o id, na folha de CSS ou na "head" usando "#", ex: "#principal".

Por vias de regra só podemos usar um id para cada pagina e ele sempre irá sobrepor uma classe ou configuração padrão.

Classes (.)

Uma classe em CSS possui um conjunto de caracteristicas usadas por todos os elementos identificados por ela.

Se quisermos personalizar vários elementos podemos criar classes. Dentro da tag desejada, usamos o comando "class=" e damos um nome a classe.

Para configurar uma classe, na folha de CSS, chamamos o nome da classe usando um ponto, ex:".classe".

Podemos usar as configurações de classes para outros elementos dentro do conteúdo, basta enveloparmos o elemento desejado com a tag <span>.

Também podemos usar mais de uma classe dentro de uma mesma tag, ex: "cass="class class2".

Podemos adicionar uma ou mais classes no "id", ex: "id="principal" class="class3".

Pseudo-classes (:)

Usamos a pseudo-classe para adicionar estados especiais dentro de de um elemento, ela é uma palavra-chave adicionada após ":" em folha de CSS, como por exemplo, ":houver" ou ":visited".

Pseudo-elemento (::)

Um pseudo-elemento é usado quado queremos adicionar um elemento dentro de outro elemento usando CSS. Para adicionar um pseudo-elemento usamos "::", ex: "::after".

Children ou filha(>)

Uma filha é uma classe dentro de outra classe, com ela podemos configurar elementos dentro de outros.
