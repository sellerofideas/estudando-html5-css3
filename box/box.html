Box inline e box em bloco

Partindo do princípio de que todo conteúdo em CSS é representado por caixas temos dois tipos de caixas.

Box inline

O box inline é usado quando queremos trabalhar com uma caixa dentro do conteúdo. Sua caracteristica é ocupar somente o espaço que lhe cabe sem precisar quebrar ou pular linha.

Exemplo de box inline.

Box em bloco

O box em bloco, por outro lado, já ocupa uma linha inteira pulando uma linha.

Exemplo de box em bloco

Na tag "Style" dentro da head ou na folha de CSS podemos alterar as caixas dos conteúdos.

Uma dica é usar o Dev tools do navegador para alterar a pagina e mostrar em tempo real para o cliente como as mudanças ficam no site, caso o cliente goste do resultado é só implementar no código original.

Propriedades das caixas

  • Caixa de texto

    • É o lugar onde o texto é mostrado.

  • panding

    • É a área entre o texto e a borda (border).

  • Border

    • É a ária da borda da caixa.

  • Margin

    • A margem é a parte exterior a borda.

Para transformar um box inline em um box em bloco basta adicionar o comando "display:". Também podemos configurar o bloco como inline, mas com outros comandos para alterar o tamanho e formas adicionando o comando "display: inline-block".
