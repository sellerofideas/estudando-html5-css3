Fontes em CSS

Tamanho das fontes

Para configurar o tamaho das fontes em CSS é usada a tag "font-size:". Existem dois tipos de tamanhos de fontes:

  1. Medodas Absolutas

    2. Cm (centímetro), mm (milímetro), in (polegada), px (pixel), pt (ponto), pc (paica).

  2. Medidas Relativas.

    3. "Em" (Medida relativa ao tamanho da letra M maiúsculo da fonte), "ex" (Relativa ao tamanho da letra x da fonte), "rem" (Relativa a fonte configurada no body), "vw" (Relativa ao percentual da largura largura da View Port ou tamanho da tela), "vh" (relativo ao percentual da altura da view port), % (Porcentagem relativa).

    Os mais utilizados para o tamanho da fonte são o "em" e o "px".
    Normalmente 16px é o mesmo tamanho que 1em. É sempre recomendado usar em para se adequar melhor ao tamanho das telas desde um relógio até uma TV.

Peso das fontes.

Para configurar o peso usamos o comando "font-weight:". Os tipos de pesos são:

  • Lighter
  • Normal
  • Negrito (bold)
  • Extra-bold
  • Numerica de 100 á 900.

Estilo das fontes

Para usar estilo nas fontes configuramos usando "font-style:", assim podemos acessar as configurações:

  • Italico
  • Oblíquo

Para simplificar os comandos existe uma configuração chamada "shorthand" que une várias configurações em uma unica linha deixando o codigo mais enxuto, a sequência que deve ser aplicada é font-style, font-wegth, font-size, font-famyli.

    O código fica:
  • font: itálic bolder 3em 'work sans', sans-serif

Google Fonts.

Para implementar a font do Google fontes é preciso escolher a fonte desejada, copiar o codigo CSS e implementar na "head", na tag "style" ou no arquivo CSS com as configurações desejadas.
